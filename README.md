# Python Fundamentals

Python exercises and refactoring practice as part of my transition
from Civil Engineering to Backend Python development.

## Modules

- `year_validator.py` : leap year logic with pure function and tests
- `is_prime.py` : prime number checker with edge case handling
- `number_compressor.py` : concatenates 3 single digits into one integer
- `employee_record.py` : Employee class with salary calculation
- `multiple_inheritance.py` : multiple inheritance with fixed MRO bug

## Structure

- `tests/` : pytest test suite for each module

## Running tests

```bash
pip install pytest
pytest -v
```
